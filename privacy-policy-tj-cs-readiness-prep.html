Chrome Extension Privacy Policy

TJ CS Readiness Prep

Effective date: March 18, 2026

Summary

This extension adds a study coach panel to CSAwesome so a student can follow a daily TJ CS readiness roadmap, open the correct topic, and track mastery locally in the browser.

This extension does not sell user data, does not share user data with third parties, and does not execute remotely hosted code unless the final build says otherwise.

What the extension stores

  • storage: This permission is used to save roadmap progress, checklist completion, streaks, achievements, and interface preferences locally on the student's device so the coach can remember progress across sessions.

How data is used

The extension stores roadmap progress, checklist completion, streaks, achievements, and interface preferences locally using Chrome extension storage. It does not transmit this information to external servers or share it with third parties.

Host access

The extension runs only on the sites required for its core functionality:

  • No explicit site access is declared.

Remote code

The extension packages its code locally and does not execute remotely hosted code based on the uploaded README.

Third-party sharing

The extension does not sell, transfer, or share user data with third parties unless the final build documentation says otherwise.

Changes

If the extension's data practices change in a future release, this policy should be updated before that version is distributed.

Privacy questions can be directed to the publisher contact listed on the Chrome Web Store item.
