# TJ CS Readiness Prep Privacy Policy

Last updated: 2026-03-18

TJ CS Readiness Prep is a Chrome extension that adds a study coach panel to CSAwesome so a student can follow a daily TJ CS readiness roadmap, open the correct topic, and track mastery locally in the browser.

## Extension Permissions

TJ CS Readiness Prep declares the following Chrome permission:

- `storage`

The extension runs on the following site pattern through its content script matches:

- `https://runestone.academy/ns/books/published/*`

`storage` is used to save roadmap progress, checklist completion, streaks, achievements, and interface preferences locally on the student's device so the coach can remember progress across sessions.

## How Data Is Used

The extension stores roadmap progress, checklist completion, streaks, achievements, and interface preferences locally using Chrome extension storage. It does not transmit this information to external servers or share it with third parties.

## Host Access

The extension runs only on the sites required for its core functionality:

- `https://runestone.academy/ns/books/published/*`

## Remote Code

The extension packages its code locally and does not execute remotely hosted code based on the uploaded project materials.

## Third-Party Sharing

The extension does not sell, transfer, or share user data with third parties unless the final build documentation says otherwise.

## Chrome Web Store Disclosure Summary

- website content is accessed: yes, only on supported Runestone pages
- webpage content is transmitted for analysis: no
- user activity within the extension is stored: yes, local roadmap progress and preferences only
- data is stored on the device: yes
- data is sold to third parties: no
- data is used for advertising or marketing: no
- data is used for creditworthiness or other eligibility determinations: no
- data is sent to a third-party remote service by default: no

## Changes To This Policy

If the extension's data practices change in a future release, this policy should be updated before that version is distributed.
